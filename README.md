# IW8 - GSC

Decompiled GSC dumps and custom scripts for Call of Duty: Modern Warfare / Warzone (IW8).

## Structure

- **Dumps/** — Decompiled `.gsc` scripts per build version (1.20.4 – 1.64.5)
- **Scripts/** — Custom GSC scripts (SOON)
